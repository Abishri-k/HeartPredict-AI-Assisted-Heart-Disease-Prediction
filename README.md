# ❤️ HeartPredict: AI-Assisted Heart Disease Prediction Using Machine Learning and Generative AI

## 📌 Overview
HeartPredict is an AI-assisted heart disease prediction system developed using Python and Google Colab. The project applies machine learning techniques to predict the risk of heart disease from patient health data. It demonstrates a complete end-to-end data science workflow, including ETL, exploratory data analysis (EDA), predictive modeling, model evaluation, SHAP explainability, and Generative AI-assisted development.

## ✨ Features
- Data preprocessing using an ETL pipeline
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine Learning model training
- Model evaluation
- SHAP-based Explainable AI
- AI-assisted development using Generative AI
- Implemented in Python using Google Colab

## 🛠 Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SHAP

## 📂 Repository Contents
- `Heart_Disease.ipynb` – Complete project notebook
- `heart_disease_dataset.csv` – Dataset used for model training

## 🚀 How to Run
1. Clone this repository.
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Install the required Python libraries.
4. Run all cells sequentially.
